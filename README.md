# Portfólio Pessoal - Daniel Germano

Meu portfólio profissional e técnico independente, projetado para consolidar e apresentar minhas competências práticas, projetos de software, desenvolvimentos em sistemas embarcados e conquistas acadêmicas.

O site possui um design moderno com uma identidade visual marcante em **Vermelho, Branco e Preto**, efeitos de vidro (*glassmorphism*), transições suaves e responsividade completa para dispositivos móveis e desktops.

## 🛠️ Tecnologias Utilizadas

*   **Estrutura:** HTML5 Semântico
*   **Estilização:** CSS3 Avançado (Vanilla CSS) com variáveis CSS e micro-animações
*   **Lógica de Interação:** JavaScript Nativo (Vanilla JS)
*   **Banco de Dados Local:** JSON estático para carregamento dinâmico de mídias e certificados

## 📌 Funcionalidades Principais

1.  **Carregamento Dinâmico de Certificados:** Os certificados e diplomas são renderizados automaticamente a partir de um arquivo JSON centralizado (`data/certificates.json`).
2.  **Visualização Direta de Mídias:** A galeria de certificados suporta visualização direta de miniaturas de mídias e abre os documentos originais (imagens ou PDFs) nativamente em uma nova aba do navegador ao clicar.
3.  **Sistema Multi-Idioma Dinâmico (PT/EN):** Script de tradução em tempo real (`lang.js`) que altera todo o conteúdo do site sem recarregar a página, persistindo a preferência do usuário no `localStorage`.
4.  **Painel Administrativo Local (`admin.html`):** Uma interface protegida localmente que permite o cadastro dinâmico de novas conquistas e certificados, salvando-os no armazenamento local do navegador (*localStorage*) e gerando exportações de arquivos JSON prontos para integração permanente.

## 🚀 Como Executar Localmente

### Pré-requisitos
*   Qualquer navegador moderno (Chrome, Firefox, Edge, Safari, Brave ou Vivaldi).

### Execução
Basta abrir o arquivo principal da aplicação no navegador de sua preferência:
```bash
# Navegar até o diretório do projeto e abrir o arquivo index.html no navegador
xdg-open index.html
```
